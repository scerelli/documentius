# Documentius

Annotate and sign PDF documents, few and simple functions. 

[![Flathub](https://flathub.org/assets/badges/flathub-badge-en.png)](https://flathub.org)

## Installation

**Flathub (recommended)**

```sh
flatpak install flathub io.github.scerelli.Documentius
```

**From source**

Requires Python ≥ 3.10, PySide6 ≥ 6.6, PyMuPDF ≥ 1.23.

```sh
pip install -e .
documentius
```
